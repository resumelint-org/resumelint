Validates Qwen2.5-1.5B for JD requirement-extraction (call 1) and per-requirement evidence-judging (call 2) against 3 inline fixtures: music-intern, software-engineer, years-mismatch. Measures JSON reliability, token budget headroom, and latency over R repeats.

One model per tab on purpose: cycling multi-GB models in a single tab is fragile on consumer GPUs. Run each model in a fresh tab, download its report, then open a new tab for the next model.
