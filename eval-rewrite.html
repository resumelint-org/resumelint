Issue #65 · dev-only · WebGPU required · not bundled into production.

Runs the selected model against every prompt variant against every fixture under tests/fixtures/rewrite/. Deterministic rubric only — no judge model is loaded.

One model per tab on purpose: cycling multiple multi-GB models in a single browser tab is fragile on consumer GPUs (eviction-then-reload can OOM). Run each model in a fresh tab, download its report, then open a new tab for the next one.
